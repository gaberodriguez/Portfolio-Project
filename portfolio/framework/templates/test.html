Responsive Layouts

Responsive layouts are really neat. Making our page responsive is making the page adapt to differnt sizes of screen, so our material will look different and better according to what size screen or what size window the page is rendered on. We can also link out code to normalize.css file to ensure that our css code is interpredted the same across all browsers.

Making our page Responsive

We can make our page responsive by using different things in CSS, like width for the columns instead of making the width a definite with by using pixels we can set the width to a percentage so it will automatically adjust to the size of the screen/window its in. We can also use media queries to make the size of out stuff on the page change when the screen is smaller, for example we can say, when the screen is smaller than 500px we can resize the page to make it look better on a smaller screen. Bootstrap is one of the sources I used in helping me understand how this works. Bootstrap acctually is already a framework and we can dive into it and use what we need from it. This is really easy to use once you understand the basics and what you need to do to use it. I really like that this is open source so anybody could use this to help them start their project. This is somethings I will definitely use in the future with future projects and work, it is easy to use and the website offers a lot of information for you to know and get familiar with how to use Bootstrap.
